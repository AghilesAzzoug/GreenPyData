Use pinned memory to reduce data transfer in RAM.

GPU cannot access data directly from the pageable memory of the CPU. The setting, pin_memory=True can allocate the staging memory for the data on the CPU host directly and save the time of transferring data from pageable memory to staging memory (i.e., pinned memory a.k.a., page-locked memory).

For more details, see on Nvidia blog on data transfer optimization.

Noncompliant Code Example

    from torch.utils.data import DataLoader
    ...
    dataloader = utils.data.DataLoader(..., pin_memory=False, ...)
    ...

Compliant Solution

    from torch.utils.data import DataLoader
    ...
    dataloader = utils.data.DataLoader(..., pin_memory=True, ...)
    ...
