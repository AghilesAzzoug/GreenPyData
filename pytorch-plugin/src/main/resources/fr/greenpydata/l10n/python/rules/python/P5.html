Directly create tensors as torch.Tensor.

PyTorch already contains almost all methods for creating tensors. These methods are faster than using native Python or Numpy first to create tensors.

If you're using GPU (or TPU) acceleration, add the device when creating the tensor.

Noncompliant Code Example

    import torch
    import numpy as np
    ...
    tensor = torch.tensor(np.random.rand(1000, 1000))
    ...

Compliant Solution

    import torch
    ...
    tensor = torch.rand([1000, 1000])
    ...
